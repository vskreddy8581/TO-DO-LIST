# 📝 My ToDo List

A simple and responsive To-Do List web application built using **HTML, CSS, and JavaScript**. This application allows users to add, complete, filter, and delete tasks with an attractive modern UI.

## 🚀 Features

* Add new tasks
* Mark tasks as completed using checkboxes
* Filter tasks:
  * All Tasks
  * Active Tasks
  * Completed Tasks
* Delete individual tasks
* Clear all tasks
* Display total number of tasks
* Add tasks using the Enter key
* Responsive design for desktop and mobile devices

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ⚙️ How It Works

### Add Task

Users can enter a task in the input field and click the **Add** button or press **Enter** to add it to the task list.

### Complete Task

Tasks can be marked as completed using the checkbox. Completed tasks are displayed with a line-through effect and different styling.

### Filter Tasks

Users can filter tasks based on their status:

* **All** – Displays all tasks
* **Active** – Displays only incomplete tasks
* **Done** – Displays only completed tasks

### Delete Task

Each task contains a delete button that removes the task from the list.

### Clear All Tasks

The **Clear All** button removes all tasks and resets the task counter.
